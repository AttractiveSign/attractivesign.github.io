The Goal

I wanted to regain control over my data and learn how to manage a linux server securely. The project started with a simple Raspberry Pi and evolved into a dedicated mini-PC running various dockerized services.

Technical Implementation

The server runs on Ubuntu Server. All services are containerized using Docker Compose for easy backups and migration.

Services Running:

  • Nextcloud: For file syncing and contacts.
  • AdGuard Home: Network-wide ad blocking.
  • Home Assistant: To control smart home devices.
  • Nginx Proxy Manager: Handles SSL certificates and routing.

Challenges & Learnings

One of the biggest challenges was configuring the reverse proxy correctly to handle Let's Encrypt certificates automatically. I also learned a lot about firewall rules (UFW) and securing SSH access.
